[![][BADGE_PRESET_ESLINT]][LINK_PRESET_ESLINT]

## 시작하기
1. 종속성 설치
    ```bash
   # NPM
    npm install -D eslint @dungsil/eslint-config
    ```
   ```bash
   # Yarn
   yarn add -D eslint @dungsil/eslint-config
   ```
   ```bash
   # PNPM
   pnpm install -D eslint @dungsil/eslint-config
   ```
2. 설정 추가
    ```bash
    echo '{"extends": ["@dungsil"]}' > .eslintrc
    ```
